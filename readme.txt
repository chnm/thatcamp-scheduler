== Description ==

THATCamp Scheduler lets you manage event registrations using user accounts in WordPress. The plugin is optimized for WordPress multisite, so each event can have its own blog, while users can register across events without create a new user account.

Visit [THATCamp Plugins](http://thatcamp.org/plugins "THATCamp Plugins") to learn more about THATCamp Badges and other plugins by THATCamp.

== Installation ==

The best way to install THATCamp Scheduler is via the Add New link under Dashboard > Plugins.

To install THATCamp Scheduler manually, follow these steps.

1. Upload the `thatcamp-scheduler` directory to `/wp-content/plugins/`
2. Activate THATCamp Scheduler through the WordPress 'Plugins' menu
3. Visit Dashboard > TC Scheduler to start managing registrations for your event.
